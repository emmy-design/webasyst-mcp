# Быстрый старт - Webasyst MCP Server

## Что это?

Это инструмент, который позволяет управлять проектами Webasyst через искусственный интеллект Claude. Вы можете создавать приложения, плагины, просматривать конфигурацию и многое другое, используя обычный человеческий язык.

## Установка за 3 минуты

### 1. Проверьте Node.js
Откройте терминал и выполните:
```bash
node --version
```
Если версия меньше 18, скачайте новую с [nodejs.org](https://nodejs.org/)

### 2. Установите MCP сервер
```bash
cd webasyst-mcp
./install.sh
```

### 3. Настройте Claude Desktop

1. Откройте Claude Desktop
2. Перейдите в Settings -> Developer
3. В поле MCP Server Configuration вставьте:

```json
{
  "mcpServers": {
    "webasyst": {
      "command": "node",
      "args": ["/путь/к/вашему/проекту/webasyst-mcp/webasyst-mcp.js"],
      "env": {}
    }
  }
}
```

4. Перезапустите Claude Desktop

## Готово!

Теперь вы можете писать Claude команды типа:

- **"Покажи все приложения в моем Webasyst проекте"**
- **"Создай новое приложение с названием 'Каталог товаров'"**
- **"Покажи плагины для приложения shop"**
- **"Создай плагин аналитики для магазина"**

## Примеры команд

### Для дизайнеров:
- "Покажи все темы для приложения blog"
- "Создай структуру нового приложения для портфолио"
- "Какие CSS файлы есть в приложении shop?"

### Для продуктовых менеджеров:
- "Какие приложения установлены в системе?"
- "Покажи конфигурацию маршрутизации"
- "Создай плагин для интеграции с CRM"

### Для разработчиков:
- "Выполни команду обновления кэша"
- "Покажи системную конфигурацию"
- "Создай новый плагин с обработчиками событий"
- "Сгенерируй PO для приложения shop и собери MO"
- "Проверь соответствие UI/локализации проекта /var/www/site"
- "Собери релизный архив проекта /var/www/site"

## Что умеет MCP сервер

- Создание приложений и плагинов
- Просмотр структуры проекта
- Работа с конфигурацией
- Выполнение CLI команд
- Управление темами
- Безопасная работа (только локальные изменения)

## Нужна помощь?

1. Убедитесь, что вы находитесь в корневой папке проекта Webasyst
2. Проверьте, что Node.js установлен (версия 18+)
3. Перезапустите Claude Desktop после настройки

---

**Совет:** Начните с простой команды "Покажи все приложения" чтобы убедиться, что всё работает правильно!
