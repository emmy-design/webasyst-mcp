## Советы по использованию

1. **Будьте конкретными**: Указывайте точные ID приложений и плагинов
2. **Используйте понятные названия**: Это поможет в дальнейшей разработке
3. **Проверяйте результат**: После создания просматривайте структуру
4. **Экспериментируйте**: MCP сервер безопасен - все изменения только локальные
5. **Следуйте UI гайдлайнам**: Для консистентного дизайна используйте рекомендации из [UI_COMPONENTS_REFERENCE.md](UI_COMPONENTS_REFERENCE.md)
