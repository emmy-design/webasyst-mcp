# Участие в разработке

Спасибо за интерес к проекту Webasyst MCP Server! 

## История проекта

Этот проект был создан **Vlad Arkhipov** в декабре 2024 года как инструмент для упрощения разработки под фреймворк Webasyst с использованием AI-ассистентов.

## Как внести вклад

### Сообщения об ошибках

1. Проверьте, что ошибка ещё не была reported в [Issues](../../issues)
2. Создайте новый Issue с описанием:
   - Версия MCP сервера
   - Версия Node.js
   - Шаги для воспроизведения
   - Ожидаемое vs фактическое поведение

### Предложения по улучшению

1. Создайте Issue с тегом `enhancement`
2. Опишите предлагаемую функциональность
3. Объясните, какую проблему это решает

### Pull Requests

1. Форкните репозиторий
2. Создайте ветку для изменений: `git checkout -b feature/my-feature`
3. Следуйте стандартам кода проекта
4. Напишите понятное описание PR
5. Убедитесь, что код работает: `npm start`

## Стандарты кода

- ES Modules (import/export)
- Async/await для асинхронных операций
- Комментарии на русском языке для документации
- Следуйте существующему стилю кода

## Лицензия

Внося вклад в проект, вы соглашаетесь с тем, что ваш код будет распространяться под лицензией MIT.
